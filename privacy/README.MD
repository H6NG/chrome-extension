# Privacy Policy

_Last updated: December 2025_

## Overview

This Chrome extension is designed to enhance the University of British Columbia (UBC) experience by providing quick access to useful resources and tools. User privacy is a priority, and the extension is built to collect **only the minimum data required** for its intended functionality.

---

## Data Collection

This extension **does not collect personal data by default**.

Data is transmitted only when a user **voluntarily submits an issue or feedback report**.

### Data that may be collected (only if provided by the user):
- Issue or feedback message
- Email address
- Subject of that issue

---

## Data Usage

Collected data is used **exclusively** to:
- Identify and fix bugs
- Improve extension functionality
- Respond to user inquiries (if contact information is provided)

Data is **never**:
- Sold
- Shared with third parties
- Used for advertising, tracking, or profiling

---

## Remote Code & Network Requests

This extension uses network requests (`fetch`) solely to:
- Send user-submitted issue reports to an administrator endpoint
- Retrieve necessary service responses

No remote code is executed to track users or alter browsing behavior.

---

## Local Storage

The extension may store **non-personal preferences locally** using Chrome’s storage API, including:
- Language selection
- Interface settings
- Stores notes (written by user)

This data remains on the user’s device and is not transmitted externally.

---

## Tracking & Analytics

This extension:
- Does not track browsing history
- Does not monitor user activity
- Does not use cookies
- Does not use third-party analytics
- Does not fingerprint users

---

## Permissions

All requested permissions are required for core functionality and are used strictly for their intended purpose.

---

## Affiliation Disclaimer

This project is **not affiliated with, endorsed by, or officially associated with** the University of British Columbia (UBC).

---

## Changes to This Policy

This privacy policy may be updated as the extension evolves. Any changes will be reflected in this document.

---

## Contact

For privacy-related questions or concerns, please use the in-extension issue reporting feature.
